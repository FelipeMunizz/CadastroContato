
# Cadastro de Contatos

Sistema para cadastro de contatos desenvolvido para facilitar a consulta, edição de contatos



## Funcionalidades

- Cadastro de cliente
- Busca e paginação
- Niveis de Usuarios
- Implementação de sistema de redefinição de senha
